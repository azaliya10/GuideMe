Accessibility

Что уже умеет MVP

  • Распознаёт людей, машины, автобусы, светофоры, велосипеды.
  • Озвучивает угрозы и важные объекты.
  • Строит пешеходный маршрут по карте.
  • Поддерживает голосовой ввод и озвучку шагов.

Идея для финальной версии

  • Реальный режим безопасного маршрута с городскими данными.
  • Точная проверка цвета светофора и переходов.
  • Интеграция с транспортными API Бишкека.
  • Офлайн-режим для слабого интернета.
