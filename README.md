# GuideMe 

**GuideMe** — это персональный помощник для незрячих и слабовидящих людей, который делает городскую среду безопасной. Создано для **Technovation Girls 2026**.

## 🚀 Почему это круто?
* **Пешеходная навигация:** Ведет по тротуарам, а не по дорогам.
* **ИИ-зрение:** Видит машины и препятствия через камеру.
* **Голосовой интерфейс:** Управляется только голосом.
* **Вибро-отклик:** Предупреждает об опасности тактильно.

## 🛠 Технологии
Google Maps API (Walking Mode), TensorFlow.js (COCO-SSD), Web Speech API.
